Categories

Categories::releaseUrl($year)#

Returns the URL of the category that contains all the content released in that particular year.

  • $year Required, the year you want to search content for
/app/Helpers/Theme/Categories.php
use Categories;
...
Categories::releaseUrl(2010);

Categories::genreCategoryUrl($genre)#

Returns the URL of the category that contains all the content released under that particular genre.

  • $genre Required, the genre you want to search content for
/app/Helpers/Theme/Categories.php
use Categories;
...
Categories::genreCategoryUrl('Comedy');

Categories::castActorCategoryUrl($actor)#

Returns the URL of the category that contains all the content released under that particular actor.

  • $actor Required, the actor you want to search content for
/app/Helpers/Theme/Categories.php
use Categories;
...
Categories::genreCategoryUrl('John Doe');

Categories::ratingCategoryUrl($rating)#

Returns the URL of the category that contains all the content released under that particular rating.

  • $rating Required, the rating you want to search content for (this refers to movie ratings, rated R, 16+, ...)
/app/Helpers/Theme/Categories.php
use Categories;
...
Categories::ratingCategoryUrl('R');

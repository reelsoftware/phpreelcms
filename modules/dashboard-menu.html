Dashboard menu

Thanks to another package provided by Nicolas Widart called laravel-menus you can actually update the existing dashboard menu to accommodate new fields generated by your modules.

How to add new field to dashboard menu?#

We have to first define new dashboard routes. In order to do that, go to /Modules/YourModuleName/Routes/web.php file and create a route group with two middleware: admin to check if the user is an admin and auth to check if the user is logged in.

/Modules/YourModuleName/Routes/web.php
Route::middleware(['admin', 'auth'])->group(function () {    Route::get('/dashboard/testing', 'TestController@index')->name('dashboardTesting);});

Now, in order to add a new option to the menu, you have to modify the dashboard-nav instance from a service provider inside your module. To do that go to /Modules/YourModuleName/Providers/YourModuleNameServiceProvider.php (or create a new service provider) and use the dashboard-nav instance to modify the dashboard menu. Don't forget to add use Menu; to the top of the file.

use Menu;
...
public function boot(){    $this->registerTranslations();    $this->registerConfig();    $this->registerViews();    $this->loadMigrationsFrom(module_path($this->moduleName, 'Database/Migrations'));
    //This will be added    Menu::modify('dashboard-nav', function($menu)    {        //Parameter 1 = name of the route to point to        //Parameter 2 = title of the URL link        //Parameter 3 = array of data to pass to the route        //Parameter 4 = icon to pass to the menu        $menu->dropdown('New section', function ($sub) {            $sub->route('dashboardTesting', 'Dashboard testing with route', [], ['icon' => 'ni ni-image']);        });    });}

We must modify it in a service provider to make sure that the new menu option is loaded on every instance of dashboard-nav.

The above code adds a new menu option to dashboard-nav with the title New section and a URL called Dashboard testing with route that points to the route dashboardTesting.

That is just a basic example. You can find more information about how menus work on laravel-menus documentation.

Keep in mind that dashboard-nav is the name of the menu that you want to modify. You must always reference it when you want to modify the dashboard menu.
